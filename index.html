Dejalist

The open source code behind the Dejalist Android application.
download .ZIPdownload .TGZ

Have you ever wondered what exactly you should write down in your shopping list? Do you often look at everything in the supermarket to remind yourself of what you need? Dejalist can help you plan easily your daily groceries lists by simply reviewing pictures of your favorite products. There is no need for complicated input - Dejalist will help you take a picture of a product and give it a name. Building a shopping list is then just a couple of clicks away.

Download

You can download the latest version of Dejalist from Google Play.

About

This app is built during the free time of the developer for fun. It also tries to ease the task of everyday shopping planning, which is a particularly extremely high complexity problem for a typical software developer ;) This app would not work and feel the same way if it weren't for some great Android open-source projects that were used during the development.

  • Android Holo ColorPicker
  • Android open source project
  • Android Quill
  • Butter Knife
  • Cupboard for Android
  • Picasso
  • Roman Nurik's Code - Done discard
  • Roman Nurik's Code - Undobar
